# 书签清理助手 (Bookmark Cleaning Assistant)

这是一个用于管理和清理 Chrome 浏览器书签的扩展程序。它可以帮助你快速检测并清理失效链接、空文件夹以及重复的书签，让你的书签栏保持整洁。

## 主要功能

*   **🧹 检测失效书签**：自动扫描所有书签链接，识别无法访问（如 404、500 错误）的网址。
*   **📂 检测空文件夹**：在扫描失效书签的同时，自动识别并列出没有任何书签的空文件夹。
*   **🔁 检测重复书签**：查找并分组显示指向同一网址的重复书签。
*   **🗑️ 批量管理**：支持全选/多选，一键删除选中的失效书签或重复项。
*   **⚙️ 自定义设置**：可调整请求超时时间，避免因网络问题导致的误判。
*   **🎨 现代化界面**：简洁美观的 UI 设计，操作直观便捷。

## 安装说明

1.  下载本项目代码到本地。
2.  打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions/`)。
3.  打开右上角的 **“开发者模式”** 开关。
4.  点击左上角的 **“加载已解压的扩展程序”**。
5.  选择本项目所在的文件夹。

## 使用指南

1.  点击浏览器工具栏中的扩展图标打开应用。
2.  **检测失效书签 & 空文件夹**：
    *   点击“检测失效书签”按钮，程序将开始扫描。
    *   扫描过程中可以随时点击“停止检测”。
    *   扫描完成后，列表中会显示失效链接和空文件夹，勾选后点击底部“删除选中书签”即可清理。
3.  **检测重复书签**：
    *   点击“检测重复书签”按钮。
    *   程序将列出所有重复的书签组，你可以保留一个，删除其余重复项。
4.  **设置**：
    *   点击顶部导航栏的“设置”图标。
    *   可以调整网络请求的超时时间（默认为 15秒），如果网络环境较差，建议适当延长。

## 项目结构

*   `index.html`: 主界面结构
*   `index.js`: 核心业务逻辑（书签扫描、DOM 操作、事件处理）
*   `style/`: 样式文件目录
    *   `index.css`: 主样式
    *   `duplicate.css`: 重复书签页面的特定样式
*   `js/`: 其他脚本文件
*   `_locales/`: 多语言支持配置文件
*   `manifest.json`: Chrome 扩展的核心配置文件

## 注意事项

*   **数据安全**：删除操作是**不可撤销**的，建议在进行大规模清理前先导出备份书签。
*   **网络请求**：由于检测失效链接需要发送网络请求，部分网站可能会因为频繁请求而暂时拒绝访问（HTTP 429），这属于正常现象。
