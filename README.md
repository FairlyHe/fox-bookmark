# 书签清理助手 (Bookmark Cleaning Assistant)

这是一个用于管理和清理 Chrome 浏览器书签的扩展程序。它可以帮助你快速检测并清理失效链接、空文件夹以及重复的书签，让你的书签栏保持整洁。

## 主要功能

*   **🧹 检测失效书签**：自动扫描所有书签链接，识别无法访问（如 404、500 错误）的网址。
*   **📂 检测空文件夹**：在扫描失效书签的同时，自动识别并列出没有任何书签的空文件夹。
*   **🔁 检测重复书签**：查找并分组显示指向同一网址的重复书签。
*   **♻️ 书签回收站**：删除的书签会先移入回收站，支持一键恢复，给您“后悔药”。
*   **🧠 智能保留策略**：针对重复书签，支持一键“保留最早创建项，选中其余重复项”。
*   **🎯 扫描范围选择**：可按顶层文件夹选择扫描范围，避免全盘扫描。
*   **📜 扫描历史与统计**：记录每次扫描的统计数据（总数、发现、清理、耗时），方便回溯。
*   **⌨️ 快捷键支持**：提供常用操作的快捷键，提升效率。
*   **🎨 现代化界面**：
    *   **Tab 导航**：失效/重复书签独立管理。
    *   **斑马线列表**：单行紧凑显示，清晰易读。
    *   **无抖动预览**：优化的预览窗口体验。

## 安装说明

1.  下载本项目代码到本地。
2.  打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions/`)。
3.  打开右上角的 **“开发者模式”** 开关。
4.  点击左上角的 **“加载已解压的扩展程序”**。
5.  选择本项目所在的文件夹。

## 使用指南

### 1. 基础检测
*   **检测失效书签 & 空文件夹**：
    *   在首页勾选需要扫描的书签文件夹范围。
    *   点击“检测失效书签”按钮（或快捷键 `Alt + I`）。
    *   扫描完成后，结果列表支持全选或手动勾选，点击“删除选中书签”移入回收站。
*   **检测重复书签**：
    *   点击“检测重复书签”按钮（或快捷键 `Alt + D`）。
    *   使用“智能保留”按钮一键勾选多余的重复项（保留创建最早的）。

### 2. 回收站管理
*   点击顶部导航栏的 **回收站 (♻️)** 图标。
*   查看已删除的书签，支持 **“恢复选中”**（还原到原位置）或 **“清空回收站”**（永久删除）。

### 3. 历史记录
*   点击顶部导航栏的 **历史 (📜)** 图标。
*   查看过往的扫描记录和清理统计。

### 4. 设置与快捷键
*   点击顶部导航栏的 **设置 (⚙️)** 图标。
*   **超时设置**：调整网络请求超时时间（默认 15秒）。
*   **快捷键说明**：
    *   `Alt + I`: 开始失效检测
    *   `Alt + D`: 开始重复检测
    *   `Alt + Del`: 删除选中项

## 项目结构

*   `index.html`: 主界面结构
*   `index.js`: 核心业务逻辑（书签扫描、回收站、历史记录、DOM 操作）
*   `style/`: 样式文件目录
    *   `index.css`: 主样式（包含 Flex 布局、斑马线列表、预览窗口等）
    *   `duplicate.css`: 重复书签页面的特定样式
*   `js/`: 其他脚本文件
    *   `preview.js`: 预览窗口逻辑
*   `_locales/`: 多语言支持配置文件
*   `manifest.json`: Chrome 扩展的核心配置文件

## 注意事项

*   **数据安全**：虽然有回收站功能，但在执行“清空回收站”前请务必确认，该操作是**不可撤销**的。
*   **网络请求**：由于检测失效链接需要发送网络请求，部分网站可能会因为频繁请求而暂时拒绝访问（HTTP 429），这属于正常现象。
*   **版本更新**：修改 `manifest.json` 后需要在扩展管理页面点击“重载”图标才能生效。
